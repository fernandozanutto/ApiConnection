### Arquitetura
Utilizada a arquitetura MVVM, separando as classes do projeto entre Models, Views e ViewModels

### Tecnologias e Bibliotecas utilizadas
- Kotlin
- Material Design
- LiveData
- Google Maps SDK
- Glide
- Volley
- SwipeRefreshLayout
